「Claude Codeっていうのがすごいらしい。使ってみたいけど、コードが書けない自分に始められるのか？　そもそもWindowsで動くのか？」

結論から言うと、**Windowsで動きますし、プログラミング知識はゼロで始められます**。実際、私はコードが書けないままClaude Codeを毎日使っています。

ただし、始めるまでに私はいくつかつまずきました。この記事では、公式の手順と、私が実際にハマったポイント＆解決法をまとめます。

## 結論: 必要なのは3つだけ

- **Windowsのパソコン**（特別なスペックは不要。普通の家庭用PCでOK）
- **Claudeのアカウント**（有料プランが必要。Pro／Max／Team／Enterpriseのいずれか。API利用のConsoleアカウントでも可）
- **PowerShell**（Windowsに最初から入っている黒い画面。インストール不要）

かかる時間は、スムーズにいけば**15〜30分**です。

［ここに体験を書く: 自分が始めたときの状況。どのプランにしたか、なぜClaude Codeを使おうと思ったか］

## 手順1: Claudeのアカウントを用意する

まず [Claude](https://claude.ai/) のアカウントを作り、有料プランに登録します。無料アカウントではClaude Codeは使えません（2026年7月時点）。

どのプランを選べばいいかは長くなるので別記事にしますが、まず試すなら一番安いProからで大丈夫です。

## 手順2: インストール（コマンド1行コピペするだけ）

1. スタートメニューで「**PowerShell**」と検索して開く
2. 次の1行を貼り付けて Enter を押す

```
irm https://claude.ai/install.ps1 | iex
```

これだけです。「irmって何？」とか理解しなくて大丈夫です（私も分かっていません）。公式が用意してくれているインストール用の呪文だと思ってください。

［ここにスクリーンショットを挿入: PowerShellでインストールが進んでいる画面］

なお、公式によると「Git for Windows」というソフトも入れておくのが推奨とのこと。これも [公式サイト](https://git-scm.com/downloads/win) からダウンロードして、全部「次へ」で入れればOKです。

## 手順3: 起動してログインする

PowerShellで次のように打ちます。

```
claude
```

初回はログインを求められるので、画面の案内に従うとブラウザが開きます。手順1で作ったClaudeアカウントでログインすれば連携完了。**一度ログインすれば、次回からは不要**です。

［ここにスクリーンショットを挿入: 初回起動〜ログインの画面］

## 手順4: 最初の一言を打ってみる

ログインできたら、もう普通に日本語で話しかけられます。私が最初にやったのはこんな感じの指示です。

［ここに体験を書く: 最初にClaude Codeに何を頼んだか、そのとき何が起きたか］

コードを書く必要は一切ありません。「〇〇を作って」「これはどういう意味？」と日本語で打つだけです。

## まとめ: 「黒い画面」への恐怖さえ乗り越えれば、あとは日本語

Claude Codeの始め方をまとめると:

1. Claudeの有料アカウントを作る
2. PowerShellにコマンドを1行コピペ
3. `claude` と打ってログイン
4. あとは日本語で話しかけるだけ

「黒い画面（PowerShell）を開く」という一点だけが心理的ハードルですが、やることはコピペだけ。コードが書けない私でも、この環境から[ブログを1個まるごと作れました](../2026-07-11-blog-made-by-ai/)。
